My Arduino project
My project is a running line that can be used in the subway, minibuses, stations, as advertising or announcements. The main idea was to combine an Rfid module and a matrix module that could be used in schools. For example, it will work as follows: when entering the school, we will see a large screen with a running line and a scanner nearby. When student attaches his electronic ID card to the scanner, the student will see special information for him, such as timetable or the start and end time of lessons.
